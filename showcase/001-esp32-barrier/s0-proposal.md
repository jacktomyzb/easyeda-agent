| 2.4G RF | π 型匹配网络(L+C×2)+ 天线(形式待决策 ③) | P1 |
| USB/烧录 | USB-C(单取向,见"已定偏差")+ CH340C + Q1/Q2 自动下载(S8050×2)+ BOOT/RESET 按键 | P1 |
| 电源 | 5V→3.3V LDO(AMS1117 起步;若决策④选 12V 输入则加降压前级) | P2 |
| CC1101 | 形式待决策 ①(裸片+TI 参考巴伦 / 成品贴片模块)+ 433M 天线(决策③) | P2 |
| SD 卡 | microSD 自弹卡座,SPI3 | P2 |
| RS485 | 3.3V 收发器(SP3485)+ 3P 接线端子(A/B/GND)+ 偏置/终端电阻 | P2 |
| 现场接口 | K230 UART 4P 排针、磁簧 2P 端子 | P2 |
| 指示/机械 | 状态 LED(IO48 低电平点亮+15k,按固件)+ 四角 M3 | — |

**电源树**:USB VBUS(∥ 决策④的现场电源)→ +5V → LDO → 3V3 → 全部逻辑;CC1101/SD/RS485 均 3.3V。

## 已定项(不占用你的决策,但列出来备查)

- **叠层执行**:若决策②=4 层 → L2=GND 内电层,L3=3V3 铺铜(单地域,判据:无音频/功放分域需求)
- **天线区**:两个天线周边全层禁布(guardrail);2.4G 天线按 N8R8 官方板尺寸抄
- **USB-C 单取向**(已知偏差):标准库 16P 封装上正反插数据线拓扑无解(2026-07-08 实测),
  首版单取向——反插只充电不通信。要双取向需换官方同款封装,可在 v0.2 处理
- **选型**:standard-parts.json 优先;flash 8MB / 不加 PSRAM(固件无 PSRAM 需求)
- **工程**:新建 EasyEDA 工程 `esp32-barrier`(真实板不进 ceshi)

## 待你拍板的 4 个决策(选项+坑+推荐)

### ① CC1101 用裸片还是模块?
| 选项 | 优点 | 坑 |
|---|---|---|
| 裸片 + TI 参考电路 | 全芯片级一体化,量产成本最低,布局自由 | 需 26MHz 晶振 + 分立巴伦/匹配(~10 个 LC),RF 性能依赖布局,首版有调试风险 |
| 成品贴片模块(E07-M1101D 类) | RF 部分免调试,首版最稳 | 成本+占板面积,和"全芯片级"初衷相悖 |

**推荐**:裸片 + TI DN010 参考电路(433M 波长长、布局容错大,参考设计成熟可抄)。

### ② 2 层还是 4 层?
| 选项 | 优点 | 坑 |
|---|---|---|
| 4 层 | 双射频有完整 GND 参考面,回流/匹配可控;SD+RS485 布线从容 | 打样贵一点 |
| 2 层 | 便宜 | 双 RF + 芯片级匹配网络在 2 层上参考面破碎,RF 指标看运气 |

**推荐**:4 层(芯片级双射频板,2 层是在赌)。

### ③ 天线形式(装什么壳决定!)
| 选项 | 适用 | 坑 |
|---|---|---|
| 两个都板载(2.4G PCB 天线 + 433M 弹簧) | 塑料壳/裸板 | 金属道闸箱内会被屏蔽成聋子 |
| 两个都 IPEX 外引 | 金属箱(道闸现场常见) | 多 2 个连接器成本 + 装配引线 |
| 2.4G 板载 + 433M 外引 | 半金属壳,433 是主业务 | 混合,BLE 配网在金属箱内可能弱 |

**推荐**:道闸改造盒大概率金属箱 → **两个都 IPEX 外引**;若确认塑料壳再改板载省成本。

### ④ 供电:只 USB 还是加现场 12V?
| 选项 | 优点 | 坑 |
|---|---|---|
| 仅 USB 5V | 电路最简 | 现场得配 USB 电源头,道闸箱里一般只有 12/24V |
| USB + DC 12V 端子(加降压前级) | 现场直接接道闸电源,USB 只做调试 | 多一级 buck(12V→5V)约 5-6 个件 |

**推荐**:加 12V 输入(这是要装现场的真实设备,不是桌面玩具)。
